# Lo-Poli.ME

## Overview
Lo-Poli.ME is a profile picture/icon/token maker where the user can create a pixel token. 

## Technologies Used
Python, pip3, ipython, Flask, Jinja2, JavaScript, Postgres/psql, and SQLAlchemy.
